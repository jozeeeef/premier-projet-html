Définition du quad:

Le quad est un engin motorisé sur trois ou quatre roues. Il n'est pas couvert, mais peut rouler sur n'importe quel terrain. Également appelé quadricycle, c'est un engin tout-terrain.

Types de quad

Il existe actuellement une vingtaine de constructeurs de quads dans le monde. À l'exception des utilitaires, les quads sont, à leur sortie d'usine, homologués pour la conduite sur route. Le tableau ci-dessous distingue les différents types de quads.

tableau des types de quad
